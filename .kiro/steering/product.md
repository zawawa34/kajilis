# Product Overview

## 製品概要
Kajilisは、iOS（Swift）とWebバックエンド（Ruby on Rails）で構成されたハイブリッドアプリケーションプロジェクトです。現在は初期開発段階にあり、基本的なプロジェクト構造が設定されています。

## コアフィーチャー
- **iOSネイティブアプリ**: SwiftUIベースのモバイルアプリケーション
- **Rails API バックエンド**: RESTful APIを提供するRuby on Rails 8.1ベースのサーバー
- **リアルタイム通信**: Action Cable（WebSocket）によるリアルタイム機能のサポート
- **モダンなフロントエンド**: Hotwire（Turbo、Stimulus）を活用した高速なUI/UX

## ターゲットユースケース
- モバイルファーストのWebアプリケーション開発
- リアルタイム機能を必要とするアプリケーション
- iOS/Web両対応のクロスプラットフォームサービス

## 主要な価値提案
- **モダンなスタック**: 最新のRails 8.1とSwiftUIを採用
- **Spec Driven Development**: Kiro方式のスペック駆動開発によるドキュメント化された開発プロセス
- **エンタープライズレディ**: Kamalデプロイ、セキュリティツール（Brakeman、Bundler Audit）標準装備
- **開発効率**: Hotwireによる高速な開発サイクル
