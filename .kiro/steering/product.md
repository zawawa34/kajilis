# Product Overview

## 製品概要
Kajilisは、家族が楽しく効率的に家事を行うためのタスク管理・コミュニケーションツールです。iOS（Swift）とWebバックエンド（Ruby on Rails）で構成されたハイブリッドアプリケーションプロジェクトです。

**現在の開発状況**: 初期開発段階 - 基本UI構造とデザインシステム実装完了

## コアフィーチャー
- **買い物リスト**: 家族で共有できる買い物リストの作成・管理
- **献立管理**: 週間献立の計画と共有
- **家事タスク管理**: 日々の家事タスクの割り当てと進捗管理
- **iOSネイティブアプリ**: SwiftUIベースのモバイルアプリケーション
- **Rails API バックエンド**: RESTful APIを提供するRuby on Rails 8.1ベースのサーバー
- **リアルタイム通信**: Action Cable（WebSocket）によるリアルタイム機能のサポート

## ターゲットユースケース
- **家族向け家事管理**: すべての家族が使いやすい直感的なインターフェース
- **家族間コミュニケーション**: 買い物や家事の共有・調整
- **招待制**: 既存メンバーが新しいメンバーを招待し、家族グループを形成

## 主要な価値提案
- **使いやすさ最優先**: Apple Human Interface Guidelines準拠のiOSネイティブ体験
- **温かみのあるデザイン**: 家庭的な雰囲気のオレンジ系アクセントカラー
- **モダンなスタック**: 最新のRails 8.1とSwiftUIを採用
- **Spec Driven Development**: Kiro方式のスペック駆動開発によるドキュメント化された開発プロセス
- **エンタープライズレディ**: Kamalデプロイ、セキュリティツール（Brakeman、Bundler Audit）標準装備
